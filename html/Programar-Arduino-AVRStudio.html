Como programar el Arduino/Genuino UNO con el IDE ARV Studio 5

Como primer requisito se necesita tener instalado los siguientes programas:

  • AVR Studio 5
  • Arduino IDE

Partiendo de que ya se tiene un proyecto listo en el AVR Studio 5, y que ya no tiene errores de sintaxis, se procederá a la configuración del mismo para la programación de la tarjeta Arduino/Genuino UNO.

  1. Conectamos la tarjeta Arduino UNO en el puerto USB del computador, y observamos en que Puerto COM quedó asociada

    2. Para ver qué puerto usa la tarjeta Arduino conectada por cable USB: entramos a panel de control y luego Administrador de dispositivos, y allí buscamos puertos COM y LPT (en el ejemplo: COM 5 para Arduino Uno)

    Otra forma es revisando desde el entorno de Arduino IDE:

  2. En el menú principal, seleccionamos la opción Tools -> External Tools

Aparecerá una ventana para configuración:

Tomemos en cuenta que:

Ahora tenemos que:

Para localizar el archivo avrdude.exe nos dirigimos a la carpeta donde fue instalado el IDE Arduino, en el cual, típicamente, la dirección es la siguiente:

C:\Program Files\Arduino\hardware\tools\avr\bin\avrdude.exe

Luego:

copiar
-C"C:\Program Files\Arduino\hardware\tools\avr\etc\avrdude.conf" -p atmega328p -c arduino -P COM5 -b 115200 -U flash:w:"$(ProjectDir)Release\$(ItemFileName).hex":i

Expliquemos un poco este código:

  • Esto ayudará a definir los valores de los parámetros de programación de la avrdude.exe, como lo son: El puerto COM, el microcontrolador, el nombre del archivo hexadecimal, entre otros
  • El archivo avrdude.conf posee una ruta que puede variar, dicha ruta también se tendrá que buscar dentro de nuestra carpeta de instalación del IDE Arduino

Luego se especifica lo siguiente:

  • -p atmega328p: Dispositivo AVR ATmega328P
  • -c arduino: Tipo de programador Arduino
  • -P COM5: Puerto COM al que está asociada la tarjeta (Para este caso COM5 pero puede variar)
  • -b 115200: Baudrate RS232 (115200 para Arduino Uno y Arduino Mega, 57600 para Arduino Nano
  • -U flash:w:"$(ProjectDir)Release\$(ItemFileName).hex":i : Tipo de memoria, nombre y ubicación del archivo

Una vez configurados los campos, guardamos los cambios con “Apply” le damos OK y ya podemos proceder a programar nuestra placa

Escrito el programa, y corregido los errores, para grabarlo en el microcontrolador se procede de la siguiente manera:

  1. Antes de compilar el programa debemos tener el entorno en Release, por defecto, al crear un proyecto nuevo, este está en modo Debug
  2. En el menú Build seleccionamos primero: Clean Solution, seguidamente: Build Solution (esto para crear el archivo .hex a cargar en el microcontrolador)
  3. En el menú Tools seleccionamos el Nombre personalizado de la Tarjeta conectada: en este caso, llamado Arduino UNO
  4. En la ventana de Output se observaran los siguientes mensajes, que dará por concluido la programación del microcontrolador, con lo que la tarjeta estará lista para operar
