Introducción al Entorno de desarrollo integrado (IDE) Arduino e instalación de Drivers

instalación de drivers

El Software Arduino para el sistema operativo Windows 7, Windows 10, vista o XP así como también para Mac y Linux lo podemos conseguir desde el siguiente enlace:

http://arduino.cc/en/Main/Software

Desde aqui podremos descargar el IDE según nuestro sistema operativo

  • Una vez descargado el archivo descomprimimos la carpeta de Arduino en una ubicación de fácil acceso
  • Conectamos la placa Arduino Uno al PC, y, lo mas probable, es que aparezca este aviso, luego continuamos con el resto de los pasos.
  1. Vamos a Panel de control y luego Administrador de dispositivos, allí buscamos la siguiente opción:
  2. Click derecho sobre Dispositivo desconocido y luego sobre la opción Actualizar software del controlador.
  3. Ingresamos a la opción Buscar software de controlador en el equipo
  4. Examinamos y buscamos la carpeta de Arduino previamente descomprimida en el paso 1. Dentro de esa carpeta accedemos a la carpeta Drivers y le damos aceptar
  5. Una vez ubicada la carpeta de Drivers damos click sobre el botón Siguiente
  6. Recibiremos la confirmación del Puerto COM asignado, este número de Puerto COM es muy importante tenerlo en cuenta a la hora de programar.
